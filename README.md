# Abdullah4AI Homebrew Tap

## Install

```bash
brew tap Abdullah4AI/tap
brew install appledev
```

## Formulas

| Formula | Description |
|---------|-------------|
| `appledev` | Apple Developer Toolkit - App Store Connect CLI + iOS app builder in one binary |
